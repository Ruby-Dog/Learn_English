## 2.
#### A和B都是
```
both   A   and   B    +    V(復數)
```
#### 是A 或是B
```
either  A   and   B    +   V(動詞與靠近的主詞一致) (與B一致)
```
#### 不是A 也不是B
```
neither  A   nor  B    +   V(動詞與靠近的主詞一致) (與B一致)
```
#### 不是A 而是b
```
not   A   but   B  +  V(動詞與靠近的主詞一致) (與B一致)
```
